# Supermarket's Sales Analysis – Power BI

## Project Overview
This project analyzes supermarket sales data to identify customer behavior, top-selling products, and sales across cities.

## Tools Used
- Power BI
- Data Cleaning & Transformation
- DAX Measures
- Data Modeling

## ETL Process
- Extracted raw CSV data
- Cleaned null values
- Created calculated columns
- Built DAX measures for KPIs

## Key Insights
- Top-selling product, quantities and total sales of it
- Sales comparison between men and women
- Sales comparison between customer type (regular vs members)
- Sales across cities
- total sales
- All of these insights will interact with each other depending on what is selected on the dashboard, providing the opportunity to access more detailed information based on the metric you want to analyze.
